# GEUT Frontend Skills

GEUT Frontend Best Practices Skills. Helps you create simple, readable and maintainable React applications. The GEUT way.

## When to Apply

Reference these guidelines when:
- Writing new React components
- Starting a new Astro project
- Implementing data fetching
- Reviewing code for performance issues
- Refactoring existing React code
- Optimizing bundle size or load times

## How to Use

Read individual rule files for detailed explanations and code examples. Check the `rules` directory.

Each rule file contains:
- Brief explanation of why it matters
- Incorrect code example with explanation
- Correct code example with explanation
- Additional context and references
